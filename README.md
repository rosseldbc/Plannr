# # 🌿 Weekend Planner

> Wat doen we dit weekend? — Een activiteitenplanner voor Diego, Britt & de kleine.

## Wat doet het?

- Kies je stemming voor het weekend
- Toggle tussen gezinsactiviteiten (peuter-proof) of date-avond
- Haalt automatisch het weer op voor vandaag, zaterdag en zondag
- AI genereert realistische activiteiten in de regio Antwerpen/Gent
- Bewaar favoriete activiteiten voor later

## Tech

- Vanilla HTML/CSS/JS — geen framework, geen build stap
- Weer via [Open-Meteo](https://open-meteo.com/) (gratis, geen API key)
- AI suggesties via Anthropic Claude API (werkt via Claude.ai)
- Gehost op Vercel

## Lokaal gebruiken

Gewoon `index.html` openen in je browser. Voor de AI-suggesties moet je de app openen via Claude.ai.

## Deployen

Verbonden met Vercel — elke push naar `main` deployt automatisch.
